# dollar
Coin 2.0

## Building

Simply run `make`

### Dependencies
Should only depend on megaparsec. Might need mtl but that should come standard w/ Haskell
```
% cabal update
% cabal install megaparsec
```


## Usage

```
% ./dollar pixel.c0
```
